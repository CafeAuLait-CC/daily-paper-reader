## 摘要
流模型和扩散模型在多种模态中生成高质量样本；然而，由于需要对底层微分方程进行数值积分，推理过程中需要多次网络评估。流映射通过直接学习微分方程的解映射来缓解这一问题，从而实现少步采样。然而，当前方法仅限于逼近常微分方程的解映射。这些方法可用于学习随机微分方程的转移核，从而获得恢复过程边际分布（弱收敛）而非解路径（强收敛）的解映射。我们提出强随机流映射（SSFMs）作为一种新颖框架，用于学习加性噪声随机微分方程的强解映射，直接将确定性流映射推广到随机设置。此外，我们引入布朗运动的多项式近似，并证明其逐路径收敛。这些结果为扩散模型的解映射提供了无模拟训练目标。我们证明，SSFMs在图像生成上优于先前的随机流映射方法，并能实现对分子系统的少步采样。

## Abstract
Flow and diffusion models generate high-quality samples in many modalities; however, many network evaluations are required during inference due to numerical integration of an underlying differential equation. Flow maps alleviate this problem by learning the solution map of the differential equation directly, enabling few-step sampling. Yet, current methods are restricted to approximating the solution map of ODEs. These methods can be used to learn the transition kernel of an SDE, thereby obtaining a solution map that recovers the marginal distributions of the process (weak convergence) rather than the solution path (strong convergence). We propose Strong Stochastic Flow Maps (SSFMs) as a novel framework for learning the strong solution map of additive-noise SDEs, directly generalizing deterministic flow maps to the stochastic setting. Further, a polynomial approximation to Brownian motion is introduced and shown to converge pathwise. These results enable a simulation-free training objective for the solution map of diffusion models. We demonstrate that SSFMs outperform previous stochastic flow map methods on image generation and enable few-step sampling of molecular systems.

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：流模型和扩散模型基于神经微分方程，生成高质量样本但推理时需大量网络评估（NFE）。流映射通过学习ODE解映射实现少步采样，但现有方法局限于ODE。
- **问题**：随机扩散模型的核心是SDE，而当前流映射只能近似ODE解，或学习SDE的**弱解**（恢复边际分布），无法得到**强解**（逐路径收敛）。弱随机流映射（如Diamond Map、GLASS等）在生成路径相关观测时失效。
- **意义**：提出**强随机流映射（SSFMs）**，首次将确定性流映射推广到加性噪声SDE，学习**Itô映射**（强解映射），为扩散模型的少步采样提供统一框架。

## 2. 论文提出的方法论

### 核心思想
- 学习加性噪声SDE（dX_t = f(t,X_t)dt + g(t)dW_t）的**Itô映射** Ψ_{s,t}(X_s, W_{[s,t]}) → X_t，该映射依赖于初始条件和布朗路径。
- 采用**Euler-Maruyama参数化**：Ψ_{s,t} = X_s + f_{s,t}(X_s, W_{[s,t]})(t-s) + g_{s,t}(W_{[s,t]})(W_t-W_s)，其中f_{s,t}和g_{s,t}为待学习的「归一化漂移/扩散积分」。

### 关键技术
1. **布朗运动多项式近似**：定义I^{(n)}_{s,t} = ∫_s^t \tilde{P}_n((u-s)/(t-s)) dW_u，其中\tilde{P}_n是移位Legendre多项式。该近似：
   - 在α-Hölder距离下逐路径收敛（Theorem 3.4-1）；
   - 系数独立正态分布（Theorem 3.4-2）；
   - 满足闭式Chen关系（Theorem 3.4-3）：系数可在子区间组合，便于实现半群条件。
2. **训练目标**（Theorem 3.5）：由两项组成
   - **匹配损失 L_{f,g}**：约束 s=t 时系数等于SDE真值（从数据或条件得分获得）；
   - **蒸馏损失 L_D**：约束半群条件 Ψ_{s,t} = Ψ_{u,t}(Ψ_{s,u}, W_{[u,t]})，确保有限时间行为。
3. **算法流程（Algorithm 1）**：
   - 采样时间 s，从数据分布获取 X_s；
   - 小步长匹配：用一步Euler-Maruyama真值校准模型；
   - 大步长蒸馏：用EMA模型预测目标，通过半群组合计算预测，最小化均方误差。

### 特点
- 训练**无模拟**：利用扩散模型的条件得分构造真实SDE系数，无需迭代积分。
- 输入布朗路径通过多项式系数 I^{(N)}_{s,t} 传入神经网络，避免了高维路径采样。
