## 摘要
向量量化运动分词器为文本到运动生成提供了紧凑的离散接口，但大多数运动码先验将码索引视为无序的分类标签。这种观点忽略了运动码的一个关键属性：它们是物理运动的解码器绑定原型，其学习到的码本可以携带具有意义的局部运动学几何。我们通过码本诊断验证了这一属性。学习到的PartVQ分组特定码之间的距离与局部运动原型距离一致，打乱控制会消除这种一致性，用逐渐远离的邻居替换码会导致解码的运动变化单调增大。这些结果表明运动码本表现出可测量的、非随机的且与解码器因果相关的几何结构。基于这一观察，我们提出了MoGeFlow，一种通过运动码本几何生成的文本到运动模型。MoGeFlow将每个运动码帧表示为一组结构化PartVQ分组特定码嵌入，学习在这些帧状态上的文本条件连续流，并将终端状态投影回有效的运动码以进行冻结解码。这保持了离散分词化的紧凑性和有效性，同时用几何感知的码本空间生成取代了分类码预测。实验在HumanML3D和KIT-ML上设立了R-Precision的新最先进水平，在生成方法中获得了最佳的HumanML3D多模态距离和KIT-ML FID，并在基准协议下获得了MotionMillion的最佳R@1、R@2、R@3和FID。

## Abstract
Vector-quantized motion tokenizers provide a compact discrete interface for text-to-motion generation, but most motion-code priors treat code indices as unordered categorical labels. This view overlooks a key property of motion codes: they are decoder-bound prototypes of physical movement, and their learned codebooks can carry meaningful local kinematic geometry. We verify this property through codebook diagnostics. Distances between learned PartVQ group-specific codes align with local motion-prototype distances, shuffled controls remove this alignment, and replacing codes with progressively farther neighbors induces monotonically larger decoded motion changes. These results show that motion codebooks exhibit measurable, non-random, and decoder-causal geometry.   Based on this observation, we propose \textbf{MoGeFlow}, a text-to-motion model that generates through motion codebook geometry. MoGeFlow represents each motion-code frame as a structured set of PartVQ group-specific code embeddings, learns a text-conditioned continuous flow over these frame states, and projects terminal states back to valid motion codes for frozen decoding. This preserves the compactness and validity of discrete tokenization while replacing categorical code prediction with geometry-aware codebook-space generation. Experiments set new state of the art in R-Precision on HumanML3D and KIT-ML, achieve the best HumanML3D MultiModal Distance and KIT-ML FID among generated methods, and obtain the best MotionMillion R@1, R@2, R@3, and FID under the benchmark protocol.

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- **生成域切换**：将生成过程从离散索引空间转移到连续码嵌入空间（即码本几何空间），利用码本中编码的运动原型结构，终端再通过最近邻投影回到离散码，以匹配冻结解码器的输入要求。
- **结构化帧表示**：每个时间步的P个PartVQ组专用码嵌入拼接成一个结构化帧状态 $y_{1,t} = [q_{t,1}; ...; q_{t,P}] \in \mathbb{R}^{Pd}$，将整体运动码帧作为生成单位。

### 关键技术细节

#### 3.1 冻结的离散解码器接口与结构化码帧
- 使用预训练的**PartVQ运动分词器**（继承自KV-Control），将骨架划分为6个数据驱动的关节组（root, upper arms, right leg, upper neck, left leg, head）。每组拥有独立的码本（128个条目）。
- 每组编码器提取局部运动特征，量化后得到组专用码嵌入 $q_{t,p}$。MoGeFlow不扁平化这些码索引，而是将它们按时间步组合成结构化帧 $Y_1 \in \mathbb{R}^{T \times Pd}$。

#### 3.2 运动码本几何诊断（验证前提）
- **度量对齐**：计算组内码嵌入距离与对应局部运动原型距离的Spearman相关系数 $\rho_p$。六个组的平均相关系数为0.821，打乱后降至-0.026，证明几何可测量且非随机。
- **解码器因果性**：将码替换为近/中/远邻，测量解码运动变化（整体、目标组、局部）。替换距离越大，运动变化单调增大，证明码本距离解码器相关。

#### 3.3 从分类码预测到码本空间流
- 标准先验：$p(Z|c) = \prod p(z_{\pi_j}|z_{\pi<j},c)$，基于索引的交叉熵损失。
- MoGeFlow：定义嵌入式端点分布 $\mu_1(\cdot|c) = (F_\phi)_\# p_{\text{data}}(\cdot|c)$，其中 $F_\phi(x)=Y_1$ 是冻结分词器映射。在码嵌入空间上学习文本条件流，终端通过投影 $\Pi_E$ 取最近邻离散码。

#### 3.4 文本条件流匹配
- 定义ODE：$\frac{dY_\tau}{d\tau} = v_\theta(Y_\tau, \tau, h(c))$，$\tau \in [0,1]$。
- 采用整流路径 $\alpha(\tau)=\tau, \sigma(\tau)=1-\tau$，插值 $Y_\tau = (1-\tau)Y_0 + \tau Y_1$，目标速度 $u_\tau = Y_1 - Y_0$。
- 损失函数：$L_{\text{flow}} = \mathbb{E}_{(x,c),Y_0,\tau}\left[ \frac{1}{\sum m_t} \sum m_t \|v_\theta(Y_\tau,\tau,h(c))_t - u_{\tau,t}\|_2^2 \right]$，仅对有效帧计算。
- 向量场网络：文本条件Transformer，每个帧作为令牌，输入投影 $W_{\text{in}}$，输出 $W_{\text{out}}$。

#### 3.5 终端投影、目标与采样
- 终端投影：对连续输出 $\tilde{Y}_1$ 按组分拆，每个组分投影到对应冻结码本的最近邻。
- 分类器自由引导（CFG）：在速度空间 $\tilde{v} = v_\theta(\cdot, \tau, h(\emptyset)) + s [v_\theta(\cdot, \tau, h(c)) - v_\theta(\cdot, \tau, h(\emptyset))]$。
- 采样：从高斯噪声 $Y_0\sim N(0,I)$ 开始，欧拉积分 $\tau=0\to1$，最后投影解码。
