## 摘要
向量量化运动分词器为文本到运动生成提供了紧凑的离散接口，但大多数运动码先验将码索引视为无序类别标签。这种观点忽略了运动码的一个关键属性：它们是物理运动的解码器绑定原型，其学习到的码本可以携带具有局部意义的运动学几何结构。我们通过码本诊断验证了这一属性。学习到的PartVQ组特定码之间的距离与局部运动原型距离一致，打乱控制消除了这种一致性，并且用渐进更远的邻居替换码会导致解码后的运动变化单调增大。这些结果表明运动码本表现出可测量、非随机且解码器因果的几何结构。基于这一观察，我们提出了MoGeFlow，一种通过运动码本几何生成文本到运动的模型。MoGeFlow将每个运动码帧表示为结构化的一组PartVQ组特定码嵌入，学习这些帧状态上的文本条件连续流，并将终端状态投影回有效的运动码以进行冻结解码。这保留了离散分词化的紧凑性和有效性，同时用几何感知的码本空间生成取代了类别码预测。实验在HumanML3D和KIT-ML上实现了R-Precision的最新结果，在生成方法中获得了最佳的HumanML3D多模态距离和KIT-ML FID，并在基准协议下获得了最佳的MotionMillion R@1、R@2、R@3和FID。

## Abstract
Vector-quantized motion tokenizers provide a compact discrete interface for text-to-motion generation, but most motion-code priors treat code indices as unordered categorical labels. This view overlooks a key property of motion codes: they are decoder-bound prototypes of physical movement, and their learned codebooks can carry meaningful local kinematic geometry. We verify this property through codebook diagnostics. Distances between learned PartVQ group-specific codes align with local motion-prototype distances, shuffled controls remove this alignment, and replacing codes with progressively farther neighbors induces monotonically larger decoded motion changes. These results show that motion codebooks exhibit measurable, non-random, and decoder-causal geometry.   Based on this observation, we propose \textbf{MoGeFlow}, a text-to-motion model that generates through motion codebook geometry. MoGeFlow represents each motion-code frame as a structured set of PartVQ group-specific code embeddings, learns a text-conditioned continuous flow over these frame states, and projects terminal states back to valid motion codes for frozen decoding. This preserves the compactness and validity of discrete tokenization while replacing categorical code prediction with geometry-aware codebook-space generation. Experiments set new state of the art in R-Precision on HumanML3D and KIT-ML, achieve the best HumanML3D MultiModal Distance and KIT-ML FID among generated methods, and obtain the best MotionMillion R@1, R@2, R@3, and FID under the benchmark protocol.

#### 1. 论文的核心问题与整体含义

-   **研究动机**：文本到运动生成旨在从自然语言描述中合成逼真的人体运动。当前主流的“基于令牌”方法，通常使用向量量化（Vector-Quantized, VQ）分词器将连续运动压缩为离散代码序列。然而，这些方法大多只将运动代码视为无序的类别标签（Categorical Labels），预测下一个代码的索引。
-   **核心观察**：本文作者认识到这一“类别视角”是不完整的。与任意的类别标签不同，运动代码是物理运动模式的“原型”（Prototypes），其学习得到的嵌入空间（Code Embedding Space）蕴含了有意义的局部运动学几何结构。即，在嵌入空间中距离相近的代码，解码后对应的运动模式也更为相似。
-   **核心问题**：是否能利用运动码本的这种“几何属性”来改进生成过程，而不是仅仅进行离散的类别预测？
-   **整体含义**：本文挑战了现有方法的范式，提出离散性应保留在解码器接口（Decoder Interface），而生成过程本身可以在连续的、具有几何意义的码本嵌入空间中进行。这兼顾了离散表示的紧凑有效性和连续生成的几何感知能力。

#### 2. 论文提出的方法论

-   **核心思想**：不再将运动代码生成视为一个离散分类任务，而是在一个冻结的运动码本（Freezed PartVQ Codebook）的连续嵌入空间中，学习一个文本条件驱动的连续流（Flow），最后将生成的连续状态“投影”（Project）回最近的、有效的离散代码以供解码器使用。
-   **关键技术细节**：
    1.  **结构化运动码帧（Structured Motion-Code Frame）**：利用一个预训练的 PartVQ 分词器（将骨架划分为6个数据驱动的局部组），将运动序列映射为 `T x P x d` 的嵌入张量。MoGeFlow 将同一时间步（`t`）的 `P` 个组代码拼成一个“帧状态”（Frame State）`y1_t`。生成单元是整个帧状态，而非单个组代码。
    2.  **运动码本几何诊断（Motion Codebook Geometry Diagnostics）**：
        -   **度量对齐**：计算码本嵌入距离与对应局部运动原型距离间的斯皮尔曼相关系数（`ρ`），并与打乱代码与原型关联的对照组进行比较。
        -   **解码器因果性**：将序列中的某个代码替换为码本中距离较远的邻居，并测量解码后运动的差异。替换代码距离越远，运动变化越大。
    3.  **码本空间流匹配（Codebook-Space Flow Matching）**：
        -   定义一个文本条件 ODE：`dYτ/dτ = vθ(Yτ, τ, h(c))`，其中 `Yτ` 是连续帧状态，`h(c)`是文本特征。
        -   使用条件流匹配（CFM）训练：对干净端点 `Y1` 和噪声 `Y0` 进行线性插值 `Yτ = (1-τ)Y0 + τY1`，目标速度 `uτ = Y1 - Y0`。损失函数 `L_flow` 让网络 `vθ` 预测这个速度。
        -   与类别预测不同，该损失在连续码本空间提供向量级的监督信号。
    4.  **终端投影与解码（Terminal Projection & Decoding）**：
        -   采样时，ODE 积分得到的终端状态 `˜Y1` 不会精确落于离散码本上。
        -   将 `˜Y1` 中的每个组分量投影到其对应组码本中最近邻的代码，得到有效代码序列 `Ẑ`。
        -   最后，冻结的解码器 `Dϕ` 将 `Ẑ` 解码为最终运动 `ˆx`。
-   **公式/算法流程（文字描述）**：
    1.  运动 `x` 通过冻结的 PartVQ 分词器 `Fϕ` 得到结构化帧状态 `Y1`。
    2.  采样高斯噪声 `Y0` 和时间 `τ`。
    3.  构建线性路径 `Yτ = (1-τ)Y0 + τY1`，目标速度为 `uτ = Y1 - Y0`。
    4.  训练文本条件向量场网络 `vθ`，最小化其输出与 `uτ` 的均方误差。
    5.  推理时，从噪声 `Y0` 开始，用训练好的 `vθ` 积分 ODE 从 `τ=0` 到 `τ=1`。
    6.  对得到的连续终端状态 `˜Y1`，进行最近邻投影得到离散代码序列 `Ẑ`。
    7.  用冻结的运动解码器 `Dϕ` 将 `Ẑ` 解码生成最终运动。
