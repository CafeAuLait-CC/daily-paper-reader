## 摘要
实现完全自动化、物理合理的3D运动合成是图形学和生成式AI的核心目标。然而，配置复杂环境力场仍完全依赖人工专家干预，这为大规模仿真数据生成造成了严重瓶颈。现有自动化方法主要聚焦于材料优化，在应用于复杂度更高的力场优化空间时存在显著的模态缺口和技术缺陷：朴素的大语言模型缺乏底层仿真反馈，导致严重的物理不准确性；而传统的分数蒸馏采样则存在梯度缓慢、局部最优陷阱以及在数学上无法动态切换离散力场的问题。为此，我们提出PhysAgent——首个仿真器在环的多智能体框架，它利用多模态输入实现自动化、物理合理的4D合成。通过将内禀材料与外源动力学解耦，PhysAgent利用配备外化力场技能模块的语义智能体来掌握仿真规则并生成有效的初始状态。随后，基于轨迹的多智能体反馈驱动的优化智能体利用视觉基础模型从渲染帧中提取稠密点轨迹。通过将这些显式运动轨迹转换为结构化文本描述符，智能体利用大语言模型的常识推理能力执行零样本宏观跃迁，有效逃离局部最优并动态切换离散力场。大量实验表明，PhysAgent能从任意多模态提示中快速生成稳定、多样化的物理场景，在生成多样性和物理准确性上显著优于现有基准方法。

## Abstract
Achieving fully automated, physically plausible 3D motion synthesis is a core objective in graphics and generative AI. However, configuring complex environmental force fields still relies entirely on manual expert intervention, creating a severe bottleneck for large-scale simulation data generation. Existing automated methods primarily focus on material optimization and exhibit severe modality gaps and technical flaws when applied to the vastly more complex force field optimization space: naive Large Language Models (LLMs) lack underlying simulation feedback, causing severe physical inaccuracies, while traditional Score Distillation Sampling (SDS) suffers from sluggish gradients, local optima entrapment, and a mathematical inability to dynamically switch discrete force fields. To address this, we propose PhysAgent, the first simulator-in-the-loop multi-agent framework that leverages multimodal inputs for automated, physically grounded 4D synthesis. By decoupling intrinsic materials from extrinsic dynamics, PhysAgent utilizes a Semantic Agent equipped with an externalized Force Field Skill module to master simulation rules and generate valid initializations. Subsequently, the Refine Agents, driven by Trajectory-Grounded Multi-Agent Feedback, leverage vision foundation models to extract dense point trajectories from rendered frames. By converting these explicit motion trajectories into structured textual descriptors, the agent harnesses LLM commonsense reasoning to execute zero-shot macroscopic leaps, effectively escaping local optima and dynamically switching discrete force fields. Extensive experiments demonstrate that PhysAgent rapidly generates stable, diverse physical scenes from arbitrary multimodal prompts, significantly outperforming existing baselines in both generation diversity and physical accuracy.

  - 传统分数蒸馏采样（SDS）依赖缓慢梯度反向传播，易于陷入局部最优，且数学上无法动态切换离散力场类型（如方向切换）。

- **研究意义**：提出第一个“仿真器在环”（simulator-in-the-loop）的多智能体框架，实现环境力场的全自动化配置，从而打通从用户意图到物理合理4D动态的完整链路，为大规模仿真数据生成、具身AI训练等应用奠定基础。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将内禀材料（intrinsic materials）与外源动力学（extrinsic dynamics）解耦，通过语义智能体初始化力场，再由细化智能体基于闭环视觉反馈迭代优化，实现零样本宏观状态跳跃以逃离局部最优。
- **关键技术细节**：
  - **语义智能体（Semantic Agent）**：配备外化的“力场技能模块”（Force Field Skill Library），该模块明确定义支持的所有力场类型（translation, scale, impulse, torque）及其数学公式和操作约束。智能体将自然语言指令编译为结构化的JSON配置。
  - **初始化融合模块**：将语义产生的力场JSON、3DGS表示的几何锚点、PhysGM提取的材料属性、全局超参数等确定性融合为完整的仿真配置JSON。
  - **正向物理仿真**：采用材料点法（MPM），运行在100³网格上，步长2e-4秒，帧间隔0.04秒。在网格更新阶段注入外力场。
  - **轨迹基视觉感知**：利用SAM 2（分割）、Depth Anything V3（深度估计）、CoTracker3（点跟踪）从渲染视频中提取稠密3D轨迹，并格式化为结构化运动信息。
  - **细化智能体（Refine Agents）**：分为规划智能体（评估运动对齐）和执行智能体（基于反馈更新力场参数）。执行更新采用LLM的全局常识推理，公式为 Θ_{k+1} = G_LLM(Θ_k, F_k, P_text, C_physics)，实现零样本宏观跃迁，无需传统梯度。
- **算法流程**：
  1. 多模态输入（图像+文本）→ 语义智能体生成初始力场JSON → 初始化融合得到Θ₀。
  2. 执行MPM仿真，渲染视频。
  3. 视觉模块提取轨迹 → 形成结构化反馈。
  4. 细化智能体判断是否成功；若不成功，规划智能体给出修正建议，执行智能体更新力场配置。

## 7. 优点

- **创新性**：首次将多智能体闭环反馈引入物理4D合成，填补了环境力场自动化的空白。
- **鲁棒性**：通过外化力场技能模块将LLM限定在合理参数空间，防止非法配置；通过轨迹反馈持续修正语义初始误差。
- **效率**：采用推理驱动的搜索而非梯度优化，避免了缓慢的反向传播；迭代上限3次，实际生成速度快。
- **通用性**：支持多种力场类型（平移、拉伸、冲量、扭矩），可处理复杂组合指令（如“先向右吹再向左更强地吹”）。
- **可扩展性**：框架理论兼容现有材料优化方法，未来可融合。

## 8. 不足与局限

- **网格分辨率限制**：当前MPM网格为100³，限制了精细局部力场的模拟（如单根细丝的抓取）。文中提到未来研究可提升分辨率或引入部分级语义。
- **迭代次数硬上限**：3次迭代在复杂多阶段场景（如“压缩后扭曲再弹回”）可能无法全局最优，存在过早终止风险。

- **实验覆盖有限**：未在真实机器人/物理引擎中验证，仅限仿真环境；未涉及与流体、布料等复杂耦合场景。
- **资源消耗不明确**：虽只用了单GPU，但未给出每轮迭代的平均时间（LLM推理+MPM仿真+渲染），难以评估实时性。
