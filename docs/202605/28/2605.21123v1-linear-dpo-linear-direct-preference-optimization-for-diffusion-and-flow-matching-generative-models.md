## 摘要
直接偏好优化（DPO）在大型语言模型的对齐中取得成功，但在文本到图像生成中仍面临挑战。现有研究局限于去噪扩散模型而忽视了流匹配，并且在将基于自然语言处理的DPO应用于基于回归的生成任务时存在目标不匹配的问题。本文通过统一的逆向时间随机微分方程框架推导出覆盖扩散和流匹配的广义DPO目标，并从梯度角度指出标准DPO目标在文本到图像生成中是次优的。因此，我们提出线性DPO，用持续线性效用函数替代激进的基于sigmoid的效用函数，并引入EMA更新的参考模型。在扩散模型（SD1.5、SDXL）和流匹配模型（SD3-Medium）上的定性和定量实验证明，我们的方法优于现有基线。

## Abstract
Direct Preference Optimization (DPO) is successful for alignment in LLMs but still faces challenges in text-to-image generation. Existing studies are confined to denoising diffusion models while overlooking flow-matching, and suffer from an objective mismatch when applying discrete NLP-based DPO to regression-based generative tasks.\ In this paper, we derive a generalized DPO objective that covers both diffusion and flow-matching via a unified reverse-time SDE framework, and point out from a gradient perspective that the standard DPO objective is suboptimal for text-to-image generation. Consequently, we propose Linear-DPO, which replaces the aggressive sigmoid-based utility function with a sustained linear utility and incorporates an EMA-updated reference model. Qualitative and quantitative experiments on diffusion models (SD1.5, SDXL) and flow-matching model (SD3-Medium) demonstrate the superiority of our approach over existing baselines.

#### **2. 论文提出的方法论**

*   **核心思想**：本文的核心贡献是`Linear-DPO`。它通过两个关键设计来改进标准DPO：
    1.  **统一理论框架**：通过将流匹配模型也纳入一个统一的逆向时间随机微分方程框架，推导出适用于扩散、分数匹配和流匹配模型的广义DPO目标。
    2.  **优化目标重构**：从梯度角度分析了标准DPO的局限性，设计了更适合回归任务的线性效用函数和动态参考模型更新机制。

*   **关键技术细节**：
    1.  **广义DPO框架**：
        *   **方法**：为流匹配模型构造一个速度场驱动的随机微分方程，使其与逆向时随机微分方程共享相同的边际分布。通过对该随机微分方程进行欧拉-丸山离散化，可以得到具有封闭形式高斯表达式的单步条件分布。
        *   **公式`L(θ)`**：最终的统一DPO目标函数形式为：
            `L(θ) = - E [ log σ( -βT λ(t) [ ( ||y_w - y_θ||² - ||y_w - y_ref||² ) - ( ||y_l - y_θ||² - ||y_l - y_ref||² ) ] ) ]`
            其中，`y`代表预测目标（噪声`ϵ`或速度`v`），`y_θ`和`y_ref`分别是策略模型和参考模型的输出。

    2.  **Linear-DPO**：
        *   **从Sigmoid到线性效用函数**：
            *   **问题诊断**：标准DPO的梯度权重`ω(ΔD_θ) = β̄ σ(β̄ΔD_θ)`（`σ`是Sigmoid函数）存在三个问题：
                1.  `β̄`在生成模型中数值过大，导致梯度剧烈波动，需要极小的学习率（如`10⁻⁸`）。
                2.  超参数`β̄`同时控制“温度”和“更新幅度”，造成调参困难。
                3.  Sigmoid的“硬”边界导致模型快速收敛（早停），但牺牲了后期对细节的持续优化能力。
            *   **解决方案**：将梯度权重中的`β̄`从外层移入Sigmoid内部：`ω'(ΔD_θ) = σ(β̄ΔD_θ)`，并引入一个平滑的线性效用函数替代Sigmoid：`u_linear(x) = 0.2x + 0.5`。然后将其值裁剪到`[η, 1]`之间，以确保即使在偏好明显分离后模型仍能进行小幅度的持续更新（`η`是一个小常数，如`1e-2`）。
            *   **新损失函数`L_Linear-DPO(θ)`**：
                `L_Linear-DPO(θ) = E [ sg(ω'(ΔD_θ)) ( ||y_w - y_θ||² - ||y_l - y_θ||² ) ]`
                其中`sg`表示停止梯度操作，确保权重`ω'`不参与梯度反向传播。
        *   **EMA参考模型更新**：
            *   **动机**：固定的参考模型在策略模型性能远超它时会阻碍进步。周期性替换又可能不稳定。
            *   **方法**：采用指数移动平均（EMA）机制，将策略模型参数的平滑移动平均作为参考模型：`θ_ref ← γ θ_ref + (1 - γ)θ`。这样参考模型会随策略模型持续、平滑地进化。
