## 摘要
我们提出了 PersonaGesture，这是一个基于扩散模型的流水线，用于针对未见演讲者的单参考协同语音手势个性化。给定目标语音和来自新演讲者的一个动作片段，该模型必须在不进行针对每个演讲者优化的前提下，合成既符合新话语又保留演讲者特定姿态选择的手势。这种设置对于虚拟形象和虚拟代理非常有用，但其难点在于参考片段混合了稳定的演讲者习惯与特定话语的轨迹。PersonaGesture 由两个关键组件组成：自适应风格注入（ASI）和隐式分布校正（IDR），旨在将时间身份证据与残差统计校正分离。风格感知器（Style Perceiver）首先将变长参考编码为紧凑的演讲者记忆令牌。ASI 通过零初始化的残差交叉注意力将这些令牌注入去噪过程，使风格证据能够影响动作形成，而不取代预训练的语音到动作先验。在此基础上，IDR 在潜空间中应用长度感知的对角仿射映射，以校正从同一参考中估计的残差通道级矩。我们在 BEAT2 和 ZeroEGGS 数据集上评估了定量指标、参考身份控制、同音频诊断、定性比较和人类偏好。实验表明，将去噪时的演讲者记忆与保守的生成后矩校正分离，在未见演讲者个性化方面优于折叠风格码、全参考注意力和单片段微调。项目地址：https://xiangyue-zhang.github.io/PersonaGesture。

## Abstract
We propose PersonaGesture, a diffusion-based pipeline for single-reference co-speech gesture personalization of unseen speakers. Given target speech and one motion clip from a new speaker, the model must synthesize gestures that follow the new utterance while retaining speaker-specific pose choices, without per-speaker optimization. This setting is useful for avatars and virtual agents, but it is hard because the reference mixes stable speaker habits with utterance-specific trajectories. PersonaGesture consists of two key components, Adaptive Style Infusion (ASI) and Implicit Distribution Rectification (IDR), to separate temporal identity evidence from residual statistic correction. A Style Perceiver first encodes the variable-length reference into compact speaker-memory tokens. ASI injects these tokens into denoising through zero-initialized residual cross-attention, enabling style evidence to affect motion formation without replacing the pretrained speech-to-motion prior. Building on this, IDR applies a length-aware diagonal affine map in latent space to correct residual channel-wise moments estimated from the same reference. Across BEAT2 and ZeroEGGS, we evaluate quantitative metrics, reference-identity controls, same-audio diagnostics, qualitative comparisons, and human preference. Experiments show that separating denoising-time speaker memory from conservative post-generation moment correction improves unseen-speaker personalization over collapsed style codes, full-reference attention, and one-clip finetuning. *   **核心问题**：如何在仅提供**一段短参考视频**（Single-Reference）的情况下，为训练阶段从未见过的演讲者（Unseen Speakers）生成既符合新语音内容，又具备该演讲者独特风格（如手势幅度、节奏习惯、空间偏好）的协同语音手势。
*   **研究背景**：现有的个性化方法通常需要针对每个新用户进行耗时的微调（Fine-tuning）或收集大量动捕数据，这在实时交互或虚拟人应用中极具挑战。此外，参考片段往往混合了“稳定的个人风格”和“特定话语的动作轨迹”，如何提取前者而忽略后者是技术难点。

### 2. 方法论：核心思想与技术细节
PersonaGesture 采用基于扩散模型（Diffusion Model）的流水线，核心思想是将**时间维度的风格注入**与**生成后的统计分布校正**进行解耦：

*   **Style Perceiver（风格感知器）**：将变长的参考动作片段编码为固定数量的“演讲者记忆令牌（Speaker-memory tokens）”。通过对比学习（Contrastive Learning）确保这些令牌捕获的是身份一致的风格特征。
*   **Adaptive Style Infusion (ASI，自适应风格注入)**：
    *   在去噪过程中，通过“零初始化残差交叉注意力”机制将风格令牌注入冻结的扩散模型骨干网络。
    *   **作用**：在手势形成初期影响其节奏、幅度和动作包络，而不破坏预训练的语音到动作的先验知识。
*   **Implicit Distribution Rectification (IDR，隐式分布校正)**：
    *   在生成潜变量后、解码为动作前，利用参考片段的通道级均值和方差，对生成的潜变量进行对角仿射变换（基于 Wasserstein-2 最优传输理论）。
    *   **长度感知收缩规则**：引入一个随参考片段长度变化的权重 $\alpha(L)$，防止在参考片段过短时出现过度校正。

### 6. 主要结论与发现
*   **解耦有效性**：将风格作为去噪时的“记忆”和生成后的“统计校正”进行分离，效果显著优于直接微调或简单的风格编码。
*   **零初始化优势**：ASI 的零初始化设计成功保留了预训练模型的生成能力，仅学习风格残差。
*   **统计校正的必要性**：IDR 能够有效对齐潜空间的边缘分布，显著提升 SFD 指标。
*   **性能领先**：在未见演讲者的 FGD 指标上，PersonaGesture 显著优于现有 SOTA 模型（如在 BEAT2 上将 FGD 从 3.176 降至 0.371）。

### 7. 优点与亮点
*   **无需微调**：实现了真正的“零样本”个性化，测试时无需参数更新，部署极其方便。
*   **理论支撑**：IDR 组件基于最优传输理论，并针对有限样本误差提出了收缩规则，具有较好的数学解释性。
*   **紧凑的瓶颈设计**：Style Perceiver 通过固定数量的令牌解决了变长参考输入的问题，并有效过滤了特定话语的干扰。

### 8. 不足与局限
*   **极短参考限制**：虽然有长度感知校正，但当参考视频仅有 1 秒时，模型仍难以准确捕获复杂的个人习惯（这是单样本学习的固有挑战）。
*   **场景限制**：研究集中在单人说话场景，未涉及多人互动或复杂的环境约束。
*   **骨干网络依赖**：性能在一定程度上受限于预训练的 VAE 和扩散模型骨干的质量。
