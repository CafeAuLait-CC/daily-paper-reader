## 摘要
文本引导的 3D 动作编辑在单人场景中已取得成功，但由于配对数据有限以及人与人之间交互的复杂性，其在多人场景中的扩展研究较少。我们引入了多人 3D 动作编辑任务，即根据源动作和文本指令生成目标动作。为了支持这一任务，我们提出了 InterEdit3D，这是一个包含人工标注的双人动作变化的新数据集，以及一个文本引导的多人动作编辑（TMME）基准。我们提出了 InterEdit，这是一种用于 TMME 的同步无分类器条件扩散模型。它引入了带有可学习标记的语义感知计划标记对齐（Semantic-Aware Plan Token Alignment），以捕捉高层交互线索，以及一种利用离散余弦变换（DCT）和能量池化的交互感知频率标记对齐（Interaction-Aware Frequency Token Alignment）策略，用于建模周期性动作动力学。实验表明，InterEdit 提高了文本与动作的一致性以及编辑保真度，实现了最先进的 TMME 性能。数据集和代码将在 https://github.com/YNG916/InterEdit 发布。

## Abstract
Text-guided 3D motion editing has seen success in single-person scenarios, but its extension to multi-person settings is less explored due to limited paired data and the complexity of inter-person interactions. We introduce the task of multi-person 3D motion editing, where a target motion is generated from a source and a text instruction. To support this, we propose InterEdit3D, a new dataset with manual two-person motion change annotations, and a Text-guided Multi-human Motion Editing (TMME) benchmark. We present InterEdit, a synchronized classifier-free conditional diffusion model for TMME. It introduces Semantic-Aware Plan Token Alignment with learnable tokens to capture high-level interaction cues and an Interaction-Aware Frequency Token Alignment strategy using DCT and energy pooling to model periodic motion dynamics. Experiments show that InterEdit improves text-to-motion consistency and edit fidelity, achieving state-of-the-art TMME performance. The dataset and code will be released at https://github.com/YNG916/InterEdit.

*   **研究背景**：虽然单人 3D 动作编辑已取得显著进展，但**多人（双人）动作编辑**仍处于起步阶段。
*   **核心挑战**：
    1.  **数据匮乏**：缺乏“源动作-目标动作-编辑指令”这种成三元组的配对数据。
    2.  **交互复杂性**：多人动作涉及复杂的时空耦合（如同步、相位对齐、接触时机等），微小的修改可能破坏整体交互的语义。
*   **研究目的**：提出首个文本引导的多人动作编辑（TMME）任务，旨在根据文本指令修改源动作，同时保留未要求修改的交互内容。

### 2. 方法论
InterEdit 采用基于扩散模型（Diffusion Model）的架构，核心思想是通过高层语义和底层频率的双重约束来引导编辑。
*   **多人类动作表示**：采用非规范化表示，包含全局关节位置、速度、局部旋转和足部接触信息。
*   **对称交错令牌聚合（Symmetric Interleaved Token Aggregation）**：为了处理双人交互中的角色切换和时间顺序，模型将两人的动作序列进行交错排列并互换角色视角，增强模型对交互关系的建模能力。
*   **语义感知计划令牌对齐（Semantic-Aware Plan Token Alignment）**：
    *   引入可学习的“计划令牌”（Plan Tokens），通过自注意力机制捕捉全局编辑意图。
    *   利用预训练的运动教师模型（Teacher Encoder）提取目标动作的语义嵌入，并使用 **InfoNCE 损失**将计划令牌与之对齐，确保编辑符合文本描述的高层语义。
*   **交互感知频率令牌对齐（Interaction-Aware Frequency Token Alignment）**：
    *   通过计算两人动作的均值（同步信号）和差值（相对信号）来构建交互信号。
    *   应用**离散余弦变换（DCT）**将信号转至频域，并进行能量池化（划分为低、中、高频带）。
    *   通过回归损失约束模型生成的频域特征，确保动作的节奏、同步性和动力学特征得到精确保留。
*   **同步无分类器引导（SCFG）**：在推理阶段同时丢弃文本和源动作条件，以获得更纯净的引导方向，减少模型对单一条件的偏见。

*   **人数限制**：目前仅限于双人交互，尚未扩展到三人或更多人的复杂群组动作编辑。
