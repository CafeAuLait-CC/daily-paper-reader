## 摘要
在交互式应用中实时生成逼真的三维人体运动是动画、仿真和人形机器人领域的关键。尽管最近的离线运动生成方法通过文本和运动学约束提供了精确控制，但缺乏交互式场景所需的推理速度。相反，现有的在线方法能够实现实时合成，但往往牺牲了可控性，或者由于有限的上下文窗口而难以处理复杂的文本语义和长期目标。在这项工作中，我们提出了ARDY，一个流式生成框架，通过实现可由在线文本提示和灵活运动学约束控制的高保真运动生成来弥合这一差距。ARDY采用混合表示，将显式根特征与潜在身体嵌入相结合，平衡了精确轨迹控制与高效生成学习。我们提出了一种两阶段自回归变换器去噪器，具有可变历史上下文，并支持对灵活、长期运动学约束的条件化。通过在大型运动捕捉数据集上进行训练，并直接以文本标签和从真实姿态采样的运动学约束为条件，ARDY原生地学习了支持在线提示和灵活长期目标的可控生成。在HumanML3D基准和大型高保真Bones Rigplay数据集上的广泛评估证明了ARDY的高运动质量和约束遵循性，验证了我们关键架构设计的有效性。最后，我们通过一个交互式演示展示了该方法的实际多功能性，该演示具有动态文本控制、多样的关键帧姿态约束、路径跟随以及通过鼠标和键盘进行的交互式运动控制。补充视频结果、代码和模型发布见https://research.nvidia.com/labs/sil/projects/ardy/。

## Abstract
Generating realistic 3D human motions in real-time within interactive applications is key for animation, simulation, and humanoid robotics. While recent offline motion generation approaches offer precise control via text and kinematic constraints, they lack the inference speed required for interactive settings. Conversely, existing online methods enable real-time synthesis but often sacrifice controllability or struggle with complex text semantics and long-horizon goals due to limited context windows. In this work, we introduce ARDY, a streaming generation framework that bridges this gap by enabling high-fidelity motion generation controllable via online text prompts and flexible kinematic constraints. ARDY employs a hybrid representation that combines explicit root features with a latent body embedding, balancing precise trajectory control with efficient generative learning. We propose a two-stage autoregressive transformer denoiser that features variable history context and supports conditioning on flexible, long-horizon kinematic constraints. By training on a large-scale motion capture dataset and being directly conditioned on text labels and kinematic constraints sampled from ground truth poses, ARDY natively learns controllable generation that supports online prompting and flexible long-horizon goals. Extensive evaluations on the HumanML3D benchmark and the large-scale, high-fidelity Bones Rigplay dataset demonstrate ARDY's high motion quality and constraint adherence, validating the efficacy of our key architectural decisions. Finally, we demonstrate the method's practical versatility through an interactive demo featuring dynamic text control, diverse keyframe pose constraints, path following, and interactive locomotion control via mouse and keyboard. Supplementary video results, code, and model releases can be found at https://research.nvidia.com/labs/sil/projects/ardy/.

## 1. 核心问题与整体含义（研究动机与背景）

- **问题**：在交互式应用（如游戏、仿真、人形机器人控制）中，需要实时生成逼真的3D人体运动，同时支持非专业的用户输入（如自然语言文本）和灵活的运动学约束（如根轨迹、全身关键帧、末端执行器位置/旋转）。但现有方法存在两难：
  - **离线方法**（如MaskControl、Diffusion-based）能实现高质量、精细控制（文本+约束），但推理速度慢，无法满足交互式实时要求。
  - **在线/自回归方法**（如DiP、CAMDM）推理快，但通常牺牲可控性：要么不支持文本，要么只支持简单约束，要么上下文窗口短，难以处理复杂语义和长期规划（如10秒后的目标点）。
- **核心目标**：同时获得两者的优点——高保真、可控（文本+灵活约束）、实时流式生成。

## 2. 方法论

### 2.1 核心思想

- **混合表示（Hybrid Representation）**：将人体运动分解为**显式全局根运动**（位置、朝向）和**潜在身体嵌入**（通过学习的分词器压缩得到）。这既允许对根轨迹进行精确显式控制（可直接覆盖），又通过紧凑潜在空间提高了生成学习效率。
- **两阶段自回归扩散去噪器**：在每次去噪步中，先预测干净的显式根运动，再以该根预测为条件预测潜在身体运动，两者交替迭代，保证根与身体的相互影响。
- **长程约束条件**：通过掩码显式运动表示来处理任意稀疏的约束（未来10秒内的目标），包括视界内和视界外的目标。模型原生支持可变长度的历史上下文（最多8秒），从而理解复杂非周期动作的全局语义。

### 2.2 关键技术细节

1. **运动分词器（Motion Tokenizer）**：
   - 采用因果Transformer（8层，隐层512维），以补丁（patch size=4帧）为单位对显式身体运动进行编码，输出潜在身体嵌入（维度L=128，使用Finite Scalar Quantization, FSQ，64个量化级别）。
   - 与补丁化的显式根运动拼接形成混合表征：`x = [m_root_patch; x_body_latent]`。
   - 解码器将全局根转换为局部根表示（角速度、线速度、Y方向高度），以减轻脚滑。训练损失包括重建损失和脚滑惩罚。
   - 默认使用FSQ变体，训练更稳定。

2. **条件化自回归生成**：
   - 模型接收：当前视窗内C个噪声混合令牌、可变长度历史（最多H个令牌）、文本嵌入（LLM2Vec从Llama-3-8B提取）、未来约束（用掩码显式表示，包括是否约束、约束值）。
   - 根约束通过直接覆盖噪声令牌的根部分实现；身体约束通过拼接全身约束特征和掩码实现。
   - 训练时随机采样各种约束类型（根关键帧、轨迹、全身关键帧、末端执行器等），并随机旋转增强。

3. **两阶段去噪器**：
   - 根Transformer先预测干净根，身体Transformer再以干净根为条件预测潜在身体。
   - 使用DDPM框架，总损失 = 混合损失(L1) + 解码体损失(L1) + 目标误差损失(L1) + 一致性损失（预测关节位置与正运动学结果之差的L2）。
   - 训练/推理默认10步扩散，可降至4步仍保持可用性。
   - 支持延迟感知重规划机制：预留B帧缓冲以隐藏推理延迟。

### 2.3 算法流程（文字说明）

1. 运动分词器训练（自监督）：将显式身体运动补丁化为序列 → 编码为潜在身体嵌入 → 与补丁化根拼接 → 解码重建身体。训练完成后固定。
2. 去噪器训练：从训练数据中随机裁剪G帧窗口，从剩余帧中取历史H（可变）和未来F约束。给定文本嵌入、带噪声的混合令牌、历史令牌、约束掩码，模型预测干净混合令牌，计算各项损失。
3. 推理：用户在线输入文本和约束。模型生成第一个G帧窗口（无历史）。之后每个重规划步：将之前预测的令牌作为历史，输入当前噪声、约束，迭代去噪（4或10步）得到新窗口。根坐标全局平移对齐。延迟缓冲确保平滑。

## 6. 主要结论与发现

- **混合表示**显著优于纯显式表示，在保持精确根控制的同时，大幅提升了运动质量和约束遵循（FID从0.065降至0.027，关键帧误差从0.136m降至0.023m）。
- **两阶段去噪设计**（根→身体）比单阶段联合预测在约束控制上提升明显（例如全身关键帧误差从0.079降至0.023）。
- **全局到局部根转换**在分词器解码器中有效减少了脚滑（脚滑率从0.303降至0.264 m/s）。
- **长生成视界（40帧）** 和**多步扩散**有利于FID和约束精度，但2-4步已可实用（FID 0.034）。
- **在HumanML3D上**，ARDY的原始输出（无测试时优化）关节位置误差仅4.15cm，远低于MaskControl无优化版的46.18cm，接近有优化版（0.45cm）。在线设置下，ARDY的视界内/外目标误差（2.48/2.92cm）远优于DiP（9.20/17.64cm），且运动质量（FID 0.09 vs 1.45）和文本跟随（R-prec 0.69 vs 0.60）大幅领先。
- **可扩展性**：模型在大规模数据集上成功学习，支持8秒历史和10秒未来约束，展现了强大的长期规划能力。

## 7. 优点

1. **设计创新**：混合表示巧妙地结合了显式根（精确控制+直接覆盖）和潜在身体（紧凑、易学习），是平衡得失的关键。
2. **架构优雅**：两阶段去噪让根与身体训练时相互调节，避免了联合预测的模糊性。
3. **原生长程约束支持**：无需额外的RL策略或测试时优化即可处理长期目标，这是在线方法中的突破。
4. **实用性**：支持可变历史长度（0~8秒），能理解复杂非周期文本；交互演示证明了实时可控性，且包含重规划机制隐藏延迟。
5. **实验全面**：在大规模专有数据和公共基准上均进行了深入消融和对比，结论可靠。
6. **开源开放**：提供代码、模型和补充视频，促进后续研究。

## 8. 不足与局限

1. **效率问题**：自回归生成显式使用所有历史帧作为上下文（最多8秒），长序列时计算和内存开销线性增长，缺乏结构化记忆压缩机制。
2. **扩散推理成本**：即使4步，仍需多次前向传播（约33ms），在更高帧率或更复杂交互场景下可能成为瓶颈。结合捷径扩散模型可进一步加速。
3. **物理缺失**：纯运动学模型，缺乏对重力、接触力、碰撞等物理动态的建模，导致偶尔出现脚滑、飘浮、抖动等伪影。未来需整合物理模拟。
4. **分词器计算成本**：训练分词器需要单张A100 400万步，耗时较长，且对补丁尺寸敏感（太小不稳定，太大损失细节）。不同分词器配置对结果影响中等。
5. **数据集依赖**：专有Bones Rigplay数据集不公开，限制了可复现性；HumanML3D上的FID/R-Precision评估因数据预处理差异略有偏差（虽已控制）。
6. **约束类型覆盖**：虽然支持多种约束，但未展示对物体交互、多人互动、复杂地形适应等场景的泛化能力。约束采样策略可能偏向平坦地面。
7. **长期记忆有限**：虽然可处理10秒未来目标，但上下文总结机制仍是简单的拼接，最长训练序列为10秒，极长任务（如1分钟）需要截断或分段处理。
