## 摘要
角色动画中的运动混合可以通过对现有示例进行插值来合成新运动。当前方法通常局限于固定的骨架拓扑，要求不同角色具有相同或几乎相同的骨骼结构。我们提出了一种新颖的框架，用于异构骨架之间的运动混合。所提出的架构结合了语义编码器和基于扩散的解码器：语义编码器提取运动状态的逐帧潜在表示，而扩散解码器则在该潜在代码的条件下重建特定角色的运动。在推理时，通过对两个输入运动的潜在表示进行插值获得混合运动。我们在Truebones Zoo数据集上使用相同和不同骨架拓扑定义的运动进行训练和评估，展示了在多种场景下实现平滑且合理的混合能力。

## Abstract
Motion blending in character animation enables the synthesis of new motions by interpolating between existing examples. Current methods are typically restricted to fixed skeleton topologies, requiring identical or near-identical skeletal structures across characters. We present a novel framework for motion blending across heterogeneous skeletons. The proposed architecture combines a semantic encoder, which extracts per-frame latent representations of the motion state, with a diffusion-based decoder, which reconstructs character-specific motion conditioned on this latent code. At inference, blended motions are obtained by interpolating the latent representations of two input motions. We train and evaluate the method on the Truebones Zoo dataset using motions defined on both same and distinct skeleton topologies, demonstrating the ability to achieve smooth and plausible blending in a variety of scenarios.

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将每个运动解耦为“语义表征”（捕捉高层运动内容）和“随机子码”（捕捉骨架特定的细节），然后在语义潜在空间中进行插值，实现混合。
- **关键组件**：
  - **语义编码器（Semantic Encoder）**：基于 AnyTop 骨干网络（包含 Enrichment Block 和 Spatio-Temporal Transformer STT backbone），输入干净运动 X0，经过 STT 后使用全局平均池化（GAP）跨关节维度得到逐帧语义潜在向量 zsem ∈ R^(N×d)，该表征与骨架拓扑无关。
  - **随机解码器（Stochastic Decoder）**：同样基于 AnyTop 骨干，接收带噪运动 Xt，通过“语义调制”（Semantic Modulation）模块以 FiLM 风格条件化于 zsem。调制公式为：  
    ˜h_{n,j} = h_{n,j} + g(h_{n,j}, z^{(n)}_sem) ⊙ φ(z^{(n)}_sem)，其中 g 是 sigmoid 门控 MLP，φ 为投影。
  - **训练目标**：采用 AnyTop 的损失函数 L = L_simple + λ_rot L_rot，其中 L_simple 是 ℓ2 重建损失，L_rot 是旋转矩阵上的测地线损失。语义编码器与解码器联合训练，无额外正则化。
- **推理时混合流程**：
  1. 对参考运动 X0^ref 和目标运动 X0^tgt，分别通过语义编码器得到 zsem^ref, zsem^tgt，并通过 DDIM 逆推得到随机子码 ˆX_T^ref, ˆX_T^tgt。
  2. 语义混合：zsem^α = LERP(zsem^ref, zsem^tgt, α)，α 为逐帧混合系数。
  3. 随机混合：在同一骨架情况下，使用 SLERP(ˆX_T^ref, ˆX_T^tgt, α)；在跨骨架情况下，目标随机子码被替换为单位高斯噪声（因为子码与骨架绑定）。
  4. 将混合后的语义码与随机子码送入随机解码器生成混合运动。
- **采样方式**：当至少有一个随机子码与输出骨架匹配时，使用 DDIM 采样；否则（如跨骨架传递）使用 DDPM 采样。

## 3. 实验设计

- **数据集**：Truebones Zoo 数据集，包含 1,219 个运动片段（约 147,178 帧），覆盖 70 种异构骨架（哺乳动物、鸟类、昆虫、恐龙、鱼、蛇等），关节数量、连接关系、命名等差异显著，非常适合跨拓扑评估。
- **实验设置**：
  - (A) 同骨架混合：两个输入运动来自同一角色，共享骨架拓扑。
  - (B) 跨骨架混合：两个输入运动来自不同角色，骨架拓扑不同。
  - (C) 运动重定向（作为特例，α=1）：将源运动转移到目标角色。
- **评测指标**：
  - FID（Fréchet Inception Distance）：使用语义编码器作为特征空间。

  - Jerk Ratio：平滑度度量，比值接近1表示动态自然。
  - IntraDivDiff：生成运动的内部多样性与真实池的差异，越低越好。
