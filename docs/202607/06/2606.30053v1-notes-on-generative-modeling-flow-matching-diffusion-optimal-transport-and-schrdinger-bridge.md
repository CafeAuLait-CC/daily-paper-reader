### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：生成模型可通过构建从简单分布 π₀（如高斯）到目标分布 π₁ 的连续路径，并学习该路径的向量场（flow matching）或随机驱动项（diffusion/bridge matching），从而通过求解 ODE/SDE 实现采样。最优传输（OT）理论提供了选择更优耦合的依据，而熵正则 OT（即薛定谔桥）进一步引入扩散项，产生更平滑、更易学习的轨迹。

- **关键技术细节**：
  - **连续性方程（CE）**：∂tpt + div(ut·pt)=0，描述了概率密度按向量场 ut 的演化。若找到 (u, p) 满足 CE 且 p₀=π₀, p₁=π₁，则流 Φ₁ 将 π₀ 推至 π₁。
  - **流匹配**：对于任意耦合 γ∈C(π₀,π₁)，定义 pt=Law((1-t)X₀+tX₁) 和 ut(x)=E[X₁-X₀|(1-t)X₀+tX₁=x]。该 (u, p) 满足 CE。训练时用神经网络通过最小化回归损失 ∥(x₁-x₀)-v((1-t)x₀+tx₁)∥² 来近似 ut。
  - **Fokker-Planck 方程（FP）**：∂tpt+div(ut·pt)= (σ²_t/2)Δpt，对应于 SDE dXt=utdt+σ_t dBt。将流匹配的解 (v, p̃) 代入，则 ut=vt + (σ²_t/2)∇log p̃ 满足 FP，从而通过 SDE 采样得到 X₁~π₁。
  - **Benamou–Brenier 公式**：W₂²(π₀,π₁)=min_{(u,p)满足CE} ∫||ut||² dpt dt，其最优解对应最优传输耦合的流匹配解。
  - **熵正则最优传输（EOT）**：min_{γ∈C(π₀,π₁)} ∫||x₀-x₁||² dγ + ε KL(γ||π₀⊗π₁)。可以转化为 min KL(γ||Kε)，其中 Kε 是 Gibbs 测度。解由 Sinkhorn 算法给出。
  - **薛定谔桥**：有两种等价定义：(1) 动态形式 min_{(u,p)满足FP} ∫||ut||² dpt dt（即 Benamou–Brenier 加 Fisher 信息惩罚）；(2) KL 最小化形式 min_{P 路径测度, (P)₀=π₀,(P)₁=π₁} KL(P||Q)，其中 Q 是参考路径测度（如白噪声 SDE）。当 Q 对应 dXt=√ε dBt 时，等价于 EOT 2ε(π₀,π₁)。
  - **最优轨迹**：对于 EOT 耦合 (x₀,x₁)，相应 SDE 为 Brownian bridge：Xt=(1-t)x₀+tx₁+√ε t(1-t)Z，其漂移项可用条件期望近似，再通过神经网络学习。

### 6. 论文的主要结论与发现

- **主要结论**：
  1. 流匹配和扩散/桥匹配均可视为基于连续性方程或Fokker-Planck方程的采样方法，其性能强烈依赖于初始耦合 γ 的选择。
  2. 最优传输耦合（最小化平方欧氏距离）产生更直的轨迹，因此用更少的 ODE 步数即可获得高精度样本。
  3. 薛定谔桥等价于熵正则最优传输，同时提供了动态形式（惩罚Fisher信息）和静态形式（最小化KL距离），与加噪的 Brownian bridge 路径自然对应。
  4. 通过 Girsanov 定理，薛定谔桥的 KL 最小化形式（路径测度）与 Benamou–Brenier 动态形式等价，建立了扩散过程与最优传输的桥梁。

- **关键发现**：最优传输耦合（尤其是熵正则版本）是连接流匹配、扩散和薛定谔桥的统一主线。迭代使用 Sinkhorn 算法（在不可直接采样 π₁ 时通过前后向交替）可收敛到熵最优耦合，而普通 rectified flow 不保证收敛到 OT 耦合。
