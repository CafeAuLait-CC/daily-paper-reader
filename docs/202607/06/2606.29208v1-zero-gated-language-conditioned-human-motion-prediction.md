## 摘要
姿态历史为三维人体运动预测提供了核心的运动学证据，但缺乏明确的高层语义指导。本文引入ZGL，一种轻量级语言条件预测器，它利用观察到的运动描述作为语义先验，同时保留强大的运动骨干网络作为动力学主要来源。我们仅渲染观察到的姿态，使用视觉-语言模型生成一句话描述，用冻结的CLIP-L文本塔编码描述，并将其投影为一小组条件令牌。这些令牌通过带有零门的紧凑交叉注意力适配器注入到基于DCT的时空Transformer中：每个适配器输出乘以一个可学习门（初始化为零），因此整个网络在初始化时数值上与仅姿态基线相同，并且仅在减少预测误差时学习使用语言。在Human3.6M上，ZGL在我们比较的代表性运动预测基线中提高了整体MPJPE。在CMUMocap上的结果进一步表明，紧凑的描述条件化可迁移至第二个基准，并为3D人体运动预测提供了实用的语义线索。

## Abstract
Pose histories provide the core kinematic evidence for 3D human motion prediction, but they lack explicit high-level semantic guidance. This paper introduces ZGL, a lightweight language-conditioned predictor that uses captions of the observed motion as a semantic prior while preserving a strong motion backbone as the main source of dynamics. We render only the observed poses, generate a one-sentence description with a vision-language model, encode the caption with a frozen CLIP-L text tower, and project it into a small set of conditioning tokens. These tokens are injected into a DCT-based spatial-temporal Transformer by compact crossattention adapters with zero gates: each adapter output is multiplied by a learnable gate initialized to zero, so the full network is numerically identical to the pose-only baseline at initialization and can learn to use language only when it reduces prediction error. On Human3.6M, ZGL improves overall MPJPE over representative motion-prediction baselines in our comparison. Results on CMUMocap further show that compact caption conditioning transfers to a second benchmark and provides a practical semantic cue for 3D human motion prediction.

## 1. 核心问题与整体含义

- **研究动机**：三维人体运动预测（HMP）主要依赖历史姿态序列提供运动学证据，但纯运动序列缺乏高层语义指导。例如，观测到“一个人向前走”与“准备转身”在局部动力学上可能相似，需要语义先验来消歧。
- **核心问题**：如何在不破坏已有强大运动骨干网络的前提下，将语言描述作为一种轻量、互补的语义线索引入HMP，且确保语言仅在有益时发挥作用，避免过拟合或干扰运动动力学。
- **整体含义**：提出一种零门控（zero-gate）机制，使语言条件预测器在初始化时与纯运动基线完全一致，仅当caption能减少预测误差时才激活语言通道，从而以极小代价提升预测精度。

## 2. 方法论

### 核心思想
- 利用视觉语言模型（VLM）为观测到的姿态生成一句话描述（仅使用观测帧，不依赖未来）。
- 用冻结的CLIP-L文本塔编码描述，投影为少量条件令牌（conditioning tokens）。
- 通过零门控交叉注意力适配器（zero-gated cross-attention adapter）将条件令牌注入DCT-based时空Transformer的每一层编码器块。
- 门控初始化为零，使得初始化时模型行为与纯姿态基线完全一致，训练时自适应学习何时使用语言。

- **文本编码与投影**：
  - 使用Qwen2.5-VL对观测姿态的骨架渲染图生成单句描述。
  - 冻结的CLIP-ViT-L/14文本塔输出768维向量。
  - 投影头：LN → GELU线性层 → LN，输出M个条件令牌（每令牌C=128维）。
  - 引入可学习空令牌（null token），训练时以概率p=0.1替换条件令牌（类似分类器无关引导），防止模型完全依赖语言。
- **零门控交叉注意力适配器**：

## 6. 主要结论与发现

1. **语言先验有效**：ZGL在Human3.6M上达到最低短时平均MPJPE（32.83 mm）和最佳总体平均（52.74 mm），优于所有对比基线。
2. **零门控初始化关键**：初始化时精确保持基线性能，避免语言分支干扰运动动力学；门控可自适应打开。
3. **紧凑条件令牌足够**：M=4时最优，说明单向量过于压缩，更多令牌无增益。
4. **跨数据集迁移**：在CMU-Mocap上同样取得最优平均MPJPE（32.17 mm），但长时1000ms弱于SPGSN。
5. **全层注入优于后层注入**：语言特征应及早作用于运动特征，而非仅在深层融合。
6. **冻结文本塔有效**：使用冻结CLIP-L优于BGE-M3和Long-CLIP-L，且比随机或零输入显著更好，说明语言语义带来的收益非源自额外参数。

## 7. 优点

- **设计简洁高效**：零门控适配器引入极少参数（+27%），且初始化零保证与基线对齐，易于训练。

- **轻量语义接口**：使用冻结CLIP文本塔，无需联合训练大语言模型，caption可预计算，推理廉价。
- **系统消融**：逐一验证注入位置、编码器、令牌数量，为设计提供实证依据。
