## 摘要
生成逼真且符合物理规律的 3D 人机交互（HOI）仍然是动作生成中的一个关键挑战。一个主要原因是仅凭文字难以描述这些物理约束。为了解决这一局限性，我们提出了一种新范式：从易于获取的 2D 图像中提取丰富的交互先验。具体而言，我们引入了 ViHOI，这是一个新颖的框架，使基于扩散的生成模型能够利用来自 2D 图像的丰富的、特定于任务的先验，以提高生成质量。我们利用大型视觉语言模型（VLM）作为强大的先验提取引擎，并采用层解耦策略来获取视觉和文本先验。同时，我们设计了一个基于 Q-Former 的适配器，将 VLM 的高维特征压缩为紧凑的先验标记（tokens），这显著促进了我们扩散模型的条件训练。我们的框架在来自数据集的动作渲染图像上进行训练，以确保视觉输入与动作序列之间的严格语义对齐。在推理过程中，它利用由文本到图像生成模型合成的参考图像，以提高对未见物体和交互类别的泛化能力。实验结果表明，ViHOI 达到了最先进的性能，在多个基准测试中优于现有方法，并展示了卓越的泛化能力。

## Abstract
Generating realistic and physically plausible 3D Human-Object Interactions (HOI) remains a key challenge in motion generation. One primary reason is that describing these physical constraints with words alone is difficult. To address this limitation, we propose a new paradigm: extracting rich interaction priors from easily accessible 2D images. Specifically, we introduce ViHOI, a novel framework that enables diffusion-based generative models to leverage rich, task-specific priors from 2D images to enhance generation quality. We utilize a large Vision-Language Model (VLM) as a powerful prior-extraction engine and adopt a layer-decoupled strategy to obtain visual and textual priors. Concurrently, we design a Q-Former-based adapter that compresses the VLM's high-dimensional features into compact prior tokens, which significantly facilitates the conditional training of our diffusion model. Our framework is trained on motion-rendered images from the dataset to ensure strict semantic alignment between visual inputs and motion sequences. During inference, it leverages reference images synthesized by a text-to-image generation model to improve generalization to unseen objects and interaction categories. Experimental results demonstrate that ViHOI achieves state-of-the-art performance, outperforming existing methods across multiple benchmarks and demonstrating superior generalization.

生成逼真且符合物理规律的 3D 人机交互（HOI）动作是计算机动画和机器人领域的难题。现有的文本驱动生成方法主要受限于**文本描述的抽象性**：简单的文字（如“搬起箱子”）无法提供物体的精确形状、尺寸以及人机之间的空间几何约束。这导致模型在处理未见过的物体或复杂动作时，容易出现物体漂移、穿透或接触不自然等问题。本文的研究动机是利用**2D 图像作为“即插即用”的视觉先验**，通过图像中蕴含的丰富空间和几何信息来增强 3D 动作生成的真实感和泛化性。

#### 2. 方法论
ViHOI 框架由两个核心组件组成：**基于 VLM 的先验提取器**和**视觉感知 HOI 生成器**。

*   **核心思想**：将 2D 图像视为动作先验的载体，利用预训练的大型视觉语言模型（VLM）提取多模态特征，并将其注入扩散模型（Diffusion Model）中。
*   **关键技术细节**：
    *   **层解耦提取策略**：研究发现 VLM 的不同层对视觉和文本的关注度不同。ViHOI 从 Qwen2.5-VL 的**浅层（第3层）**提取视觉先验（保留几何细节），从**深层（第12层）**提取文本先验（增强语义控制）。
    *   **Q-Former 适配器**：设计了基于 Q-Former 的模块，通过可学习的查询（Query）将 VLM 输出的高维、变长特征压缩为紧凑的固定维度 Token，以便扩散模型高效处理。
    *   **训练与推理策略**：
        *   **训练时**：从地面真值（GT）动作序列中渲染出起始、中间、结束三个关键帧作为输入，确保视觉先验与 3D 动作严格对齐。

#### 6. 主要结论与发现

*   **层解耦的重要性**：从 VLM 不同深度提取特征比单一层提取效果更好，证明了视觉细节和高层语义在动作生成中具有互补性。
*   **即插即用**：ViHOI 可以作为模块集成到 MDM、CHOIS、ROG 等现有模型中，并普遍提升它们的性能。

#### 7. 优点
*   **范式创新**：提出“图像即动作先验”的新思路，巧妙地利用了 T2I 模型和 VLM 的强大能力解决 3D 稀缺数据问题。
*   **解耦设计**：通过层解耦和 Q-Former 适配器，解决了多模态特征对齐和维度冗余的问题。
*   **泛化性强**：不依赖于特定物体的几何模型，对未知类别表现出极佳的鲁棒性。
*   **实用性**：作为插件模块，易于集成到现有的扩散模型管线中。

#### 8. 不足与局限
*   **推理延迟**：由于依赖大型 VLM 和 T2I API，推理时间（约 8 秒）远高于纯文本驱动的模型，难以满足实时交互需求。
*   **手部细节缺失**：受限于现有 HOI 数据集的标注质量，模型无法生成精细的手指运动（如抓握细节）。
*   **对 T2I 模型的依赖**：如果 T2I 模型生成的图像在物理逻辑上存在严重错误（如人体结构畸形），可能会误导动作生成，尽管实验证明模型对此有一定的鲁棒性。
