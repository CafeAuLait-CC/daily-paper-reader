## 摘要
文本生成动作（T2M）正逐渐成为动画和交互式虚拟形象的实用工具。然而，在保持整体动作连贯性的同时修改特定身体部位仍然具有挑战性。现有方法通常依赖于繁琐的高维关节约束（例如轨迹），这阻碍了用户友好的迭代优化。为了解决这一问题，我们提出了模块化身体部位相位控制（Modular Body-Part Phase Control），这是一个即插即用的框架，通过紧凑的、基于标量的相位接口实现结构化的局部编辑。通过将身体部位的潜空间动作通道建模为由振幅、频率、相移和偏移量表征的正弦相位信号，我们提取了捕捉特定部位动态的可解释代码。随后，一个模块化的 Phase ControlNet 分支通过残差特征调制注入该信号，从而将控制与生成骨干网络无缝解耦。在基于扩散模型和流模型的实验表明，我们的方法对动作幅度、速度和时序提供了可预测且细粒度的控制。它在保持全局动作连贯性的同时，为可控 T2M 生成提供了一种实用的范式。项目主页：https://jixiii.github.io/bp-phase-project-page/

## Abstract
Text-to-motion (T2M) generation is becoming a practical tool for animation and interactive avatars. However, modifying specific body parts while maintaining overall motion coherence remains challenging. Existing methods typically rely on cumbersome, high-dimensional joint constraints (e.g., trajectories), which hinder user-friendly, iterative refinement. To address this, we propose Modular Body-Part Phase Control, a plug-and-play framework enabling structured, localized editing via a compact, scalar-based phase interface. By modeling body-part latent motion channels as sinusoidal phase signals characterized by amplitude, frequency, phase shift, and offset, we extract interpretable codes that capture part-specific dynamics. A modular Phase ControlNet branch then injects this signal via residual feature modulation, seamlessly decoupling control from the generative backbone. Experiments on both diffusion- and flow-based models demonstrate that our approach provides predictable and fine-grained control over motion magnitude, speed, and timing. It preserves global motion coherence and offers a practical paradigm for controllable T2M generation. Project page: https://jixiii.github.io/bp-phase-project-page/

*   **现有痛点**：目前的控制方法（如轨迹约束、关键帧）通常依赖高维、密集的关节坐标输入，操作繁琐且不直观；而基于文本的编辑方法又存在自然语言歧义，难以实现精确的定量调整。
*   **核心目标**：提出一种紧凑、可解释且“即插即用”的控制框架，允许用户通过简单的标量参数（振幅、频率、相位）直接操纵局部肢体动态。

#### 2. 方法论：核心思想与技术细节
论文提出了 **Modular Body-Part Phase Control** 框架，其核心思想是将复杂的运动动态简化为生物力学中的“相位”表示。
*   **身体部位相位模块 (Body-Part Phase Module)**：将人体骨骼划分为五个语义部分（左上肢、右上肢、左下肢、右下肢、躯干）。利用周期性自编码器从参考动作中提取四个关键标量参数：**振幅 (A)** 控制强度、**频率 (F)** 控制节奏、**相移 (S)** 控制时序、**偏移 (B)** 控制静态位置。
*   **相位流形编码**：将上述标量参数通过正弦嵌入（Sinusoidal Embeddings）转化为随时间变化的相位流形 $P$。
*   **Phase ControlNet**：借鉴图像领域的 ControlNet 思想，设计了一个辅助分支。它通过多层残差调制（Residual Modulation）将相位特征注入预训练的生成骨干网络（如 MotionLCM 或 Flow-matching 模型）。
*   **训练策略**：采用两阶段训练。第一阶段独立训练相位提取网络；第二阶段冻结生成骨干，仅训练相位编码器和 ControlNet 分支，并引入**相位一致性损失 (Phase Consistency Loss)** 以确保生成的动作严格遵循设定的相位参数。

#### 6. 主要结论与发现
*   **高精度控制**：该方法实现了对局部肢体动作幅度、速度和时序的精确、线性控制，且不会干扰未编辑部位的动作。

#### 7. 优点（亮点）
*   **直观的可解释性**：将复杂的运动控制转化为用户易于理解的“幅度、快慢、先后”三个维度。
*   **局部解耦性**：成功实现了身体部位间的运动解耦，例如在改变腿部步频时，上半身的挥手动作能保持自然连贯。
*   **交互友好**：支持多轮迭代编辑，用户可以基于生成的动作提取相位，修改后再重新生成。

*   **拓扑依赖**：相位提取网络依赖于特定的骨骼拓扑，若更换骨骼结构（如非人类角色）可能需要重新训练提取器。
